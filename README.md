# 屏蔽百度推广
chrome拓展：不管什么原因，如果你不想再看到百度推广，试试这个插件吧...

**源码安装方式**

`chrome 设置->更多工具->拓展程序->加载已解压的拓展程序 选择你解压之后的文件夹即可`
